Imagens e Figuras com Bootstrap

O bootstrap tem classes que tornam as imagens responsivas, fazendo com que elas se adéquem ao elemento que as engloba.

Usando a classe img-fluid: adéqua a largura ao container.

Usando a classe img-thumbnail: cria uma margem.

Usando a classe img-thumbnail e img-fluid

A classe rounded adiciona um raio às vertices.

Podemos usar a classe float-start para deixar uma imagem flutuando a esquerda de um conteúdo, e.g., de um texto. A classe me-3 significa margem no final igual a 3 unidades e, similarmente, mb-1 significa margem embaixo igual a 1 unidade.
